###### Sobre a Julia
Atuo no desenvolvimento de aplicações full stack, participando de todo o ciclo de desenvolvimento — desde a criação de interfaces até a integração com APIs e serviços back-end. Tenho foco na construção de código limpo, escalável e reutilizável, sempre priorizando boas práticas e performance.
Possuo experiência com TypeScript, React e Vue.js no front-end, além de desenvolvimento back-end com Node.js e Java, trabalhando com consumo e criação de APIs REST. Também aplico testes automatizados com Cypress para garantir a qualidade das aplicações.
